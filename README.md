# K კოეფიციენტის გამოთვლა
დავალების პირობის მიხედვით, K კოეფიციენტი გამოითვლება შემდეგნაირად:
$K = (სახელის ასოები + გვარის ასოები) / დაბადების რიცხვი$

ჩემს შემთხვევაში:
სახელი: ლია (3 ასო)
გვარი: გელაშვილი (9 ასო)
დაბადების რიცხვი: 24
**K = (3 + 9) / 24 = 0.5**
